# Quick Guide: How to Mark Reports as Resolved

## For Administrators

### Step-by-Step Process

#### 1. Find a Verified Report
- Go to Reports Management page
- Look for reports with **green "Verified & Published"** status
- These reports are eligible to be marked as resolved

#### 2. Click "Mark Resolved" Button
- Find the **"‚úÖ Mark Resolved"** button (green button)
- Located in the report card footer, next to Edit and Delete buttons
- Only visible for verified reports

#### 3. Fill Out Resolution Form

**Required Field:**
- **Admin Feedback** (minimum 10 characters)
  - Explain what was done to fix the issue
  - Be specific and clear
  - Example: "Pothole filled with asphalt on December 21, 2025. Road is now safe for travel."
  - Example: "Debris removed from roadway at 2:00 PM. Area cleaned and inspected."

**Optional Field:**
- **Evidence Photo** (up to 5MB)
  - Click the upload area or drag-and-drop
  - Upload a photo showing the resolved issue
  - Supported formats: PNG, JPG, GIF
  - Preview will appear - you can remove and re-upload if needed

#### 4. Submit Resolution
- Click **"Mark as Resolved & Notify User"** button
- Wait for the loading spinner (shows "Resolving...")
- Success modal will appear confirming the action

#### 5. Verification
- Report status changes to **"‚úÖ Issue Resolved"** with blue badge
- User automatically receives notification
- Report appears in "Resolved" filter
- You can still Edit or Delete the report if needed

## Common Questions

**Q: Can I resolve a pending or rejected report?**
A: No, only verified reports can be resolved. Verify the report first.

**Q: What if I make a mistake in the feedback?**
A: You can still edit the report after marking it resolved. Click the "Edit" button.

**Q: Is the evidence photo required?**
A: No, it's optional. However, it's recommended as it builds trust with users.

**Q: Can users see the evidence photo?**
A: Yes, the photo will be visible to the user who submitted the report.

**Q: Can I resolve multiple reports at once?**
A: No, each report must be resolved individually with specific feedback.

**Q: What if the issue isn't actually fixed yet?**
A: Don't mark it as resolved. Wait until the work is completed.

**Q: Can I undo a resolution?**
A: No automatic undo, but you can edit the report status manually if needed.
